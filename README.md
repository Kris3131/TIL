# Today I Learned

> 記錄每日成長、程式開發、語言學習與投資筆記

## The Roadmap

專注於以下四個核心面向：

* **English:** 目標是能流暢閱讀與交流。
* **Frontend Masters:** 深入研究底層邏輯。
* **Dev Notes:** 記錄實作中的「坑」與解決方案。
* **Finance:** 追蹤市場趨勢與投資策略。

## Repository Structure

```text
├── Daily_Logs/
│   └── 2026/
├── English/
├── Frontend_Masters/
├── Dev_Notes/
└── Finance/
```

## Workflow

為了平衡「輸入效率」與「知識架構」，採用以下流程：

1. **Daily Capture**：
* 每天在 `Daily_Logs/` 紀錄當下的靈感與碎片知識。
* 保持簡單、直覺，不強求格式。


2. **Weekly Refactor**：
* 每週末回顧本週的 `Daily_Logs`。
* 搬移至對應的面向目錄。

3. **Iterative Growth**：
* 定期重讀舊筆記，並根據新的理解進行修正與更新。
